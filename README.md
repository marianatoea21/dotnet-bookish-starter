# Dotnet Bookish Starter

## Overview

This repo contains a starter .NET server for use in the Bookish bootcamp exercise. Full details of this exercise are on the Swiki page.

## Developer Setup

1. If you haven't already, install .NET 10.0. There are instructions for how to do this on the Swiki setup page.
2. The recommended IDE is Jetbrains Rider, as that's what we will be using during the internship.
3. Run the code!
     - Run in development mode with hot reloading with `dotnet watch`
     - Run without hot reloading using `dotnet run`
     - You can check that the formatting of all files is acceptable using `dotnet format`
